Win32 API（也称为 Windows API）是需要直接访问 Windows 和硬件的本机 C/C++ Windows 应用程序的原始平台。 它提供一流的开发体验，而无需依赖于适用于 Windows 10) 的 UWP 应用的托管运行时环境（如 .NET 和 WinRT (）。 这使得 Win32 API 成为需要最高级别性能和直接访问系统硬件的应用程序的首选平台。

## 准备工作

下载或更新 Visual Studio 2019。 如果你尚未获得 Visual Studio 2019，则可以安装免费的 Microsoft Visual Studio Community 2019。 安装Visual Studio时，请确保选择**“使用 C++ 的桌面开发**”选项。 有关下载链接，请参阅“ [下载”](https://developer.microsoft.com/windows/downloads) 页。
