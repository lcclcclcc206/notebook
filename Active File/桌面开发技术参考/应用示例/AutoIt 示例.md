## Simple-IP-Config

显示并改变 ip 地址。

[KurtisLiggett/Simple-IP-Config: View and change your local IP addresses. (github.com)](https://github.com/KurtisLiggett/Simple-IP-Config)

Simple IP Config 是一个小型、便携的 ip 转换器实用程序，可以快速轻松地更改常见的 Windows 网络设置，而无需单击无休止的系统窗口。
