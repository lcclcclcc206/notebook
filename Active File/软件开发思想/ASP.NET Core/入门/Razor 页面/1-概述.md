此系列教程介绍了生成 Razor Pages Web 应用的基础知识。

有关面向熟悉控制器和视图的开发人员的更高级介绍，请参阅 [ASP.NET Core 中的 Razor Pages 简介](https://learn.microsoft.com/zh-cn/aspnet/core/razor-pages/?view=aspnetcore-7.0)。

如果你是 ASP.NET Core 开发新手，且不确定哪种 ASP.NET Core Web UI 解决方案最适合自己的需要，请参阅[选择 ASP.NET Core UI](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/choose-web-ui?view=aspnetcore-7.0)。

本系列包括以下教程：

1. [创建 Razor Pages Web 应用](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/razor-pages/razor-pages-start?view=aspnetcore-7.0)
2. [向 Razor Pages 应用添加模型](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/razor-pages/model?view=aspnetcore-7.0)
3. [基架（生成）Razor Pages](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/razor-pages/page?view=aspnetcore-7.0)
4. [使用数据库](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/razor-pages/sql?view=aspnetcore-7.0)
5. [更新 Razor Pages](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/razor-pages/da1?view=aspnetcore-7.0)
6. [添加搜索](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/razor-pages/search?view=aspnetcore-7.0)
7. [添加新字段](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/razor-pages/new-field?view=aspnetcore-7.0)
8. [添加验证](https://learn.microsoft.com/zh-cn/aspnet/core/tutorials/razor-pages/validation?view=aspnetcore-7.0)

结束时，你将有一个应用来显示和管理电影数据库。
