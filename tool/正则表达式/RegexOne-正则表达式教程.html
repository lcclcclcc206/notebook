Lesson 1: 基础知识

正则表达式在从代码、日志文件、电子表格甚至文档等文本中提取信息时非常有用。虽然形式语言背后有很多理论，但是下面的课程和例子将探索正则表达式的更多实际用途，以便您能够尽快地使用它们。

在使用正则表达式时，首先要认识到的是，每个字符本质上都是一个字符，我们正在编写模式(pattern)来匹配特定的字符序列(也称为字符串)。大多数模式使用普通的 ASCII 码，包括字母、数字、标点符号和键盘上的其他符号，比如% # $@ ！，但 unicode 字符也可用于匹配任何类型的国际文本。

Lesson 1.5: 123

字符包括普通字母，也包括数字。实际上，数字0-9也只是字符，如果查看 ASCII 表，它们是按顺序列出的。

可以使用 \d 来代表0到9之间的任意字符，前面的斜杠是为了将普通的d字符区分开来。

练习：匹配数字

Task	Text	 
Match	abc123xyz	
Match	define "123"	
Match	var g = 123;

123

Lesson 2: 点元字符

点元字符可以匹配任何单个字符（字母、数字、空格等）。不过点元字符覆盖了点字符的匹配，你需要使用 \. 来匹配它。

练习：匹配通配符

Task	Text	 
Match	cat.	
Match	896.	
Match	?=+.	
Skip	abc1

...\.

Lesson 3: 匹配特定字符

有一种使用正则表达式匹配特定字符的方法，方法是在方括号中定义它们。

例如，模式 [abc] 将匹配单个a,b,c字符，而不匹配其他任何字符。

练习：配对字符

Task	Text	 
Match	can	
Match	man	
Match	fan	
Skip	dan	
Skip	ran	
Skip	pan

[cmf]an
[^drp]an

Lesson 4: 排除特定字符

使用方括号和 ^ 来排除特定字符。

练习：排除字符

Task	Text	 
Match	hog	
Match	dog	
Skip	bog

[^b]og

Lesson 5: 字符范围

在使用方括号表示法时，可以通过使用破折号表示字符范围来简化顺序字符列表中的字符匹配。

例如，模式 [0-6] 将只匹配0到6的任何数字单位，不匹配其他任何字符。同样的，[^n-p] 将只会匹配除字母n到p之外的任何单个字符。
