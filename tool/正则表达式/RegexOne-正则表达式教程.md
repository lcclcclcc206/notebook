## Lesson 1: 基础知识

正则表达式在从代码、日志文件、电子表格甚至文档等文本中提取信息时非常有用。虽然形式语言背后有很多理论，但是下面的课程和例子将探索正则表达式的更多实际用途，以便您能够尽快地使用它们。

在使用正则表达式时，首先要认识到的是，**每个字符本质上都是一个字符**，我们正在编写模式(pattern)来匹配特定的字符序列(也称为字符串)。大多数模式使用普通的 ASCII 码，包括字母、数字、标点符号和键盘上的其他符号，比如% # $@ ！，但 unicode 字符也可用于匹配任何类型的国际文本。

## Lesson 1.5: 123

字符包括普通字母，也包括数字。实际上，数字0-9也只是字符，如果查看 ASCII 表，它们是按顺序列出的。

可以使用 `\d` 来代表0到9之间的任意字符，前面的斜杠是为了将普通的d字符区分开来。

**练习：匹配数字**

```
Task	Text	 
Match	abc123xyz	
Match	define "123"	
Match	var g = 123;

123
```

## Lesson 2: 点元字符

点元字符可以匹配任何单个字符（字母、数字、空格等）。不过点元字符覆盖了点字符的匹配，你需要使用 `\.` 来匹配它。

**练习：匹配通配符**

```
Task	Text	 
Match	cat.	
Match	896.	
Match	?=+.	
Skip	abc1

...\.
```

## Lesson 3: 匹配特定字符

有一种使用正则表达式匹配特定字符的方法，方法是在方括号中定义它们。

例如，模式 `[abc]` 将匹配单个a,b,c字符，而不匹配其他任何字符。

**练习：配对字符**
```
Task	Text	 
Match	can	
Match	man	
Match	fan	
Skip	dan	
Skip	ran	
Skip	pan

[cmf]an
[^drp]an
```

## Lesson 4: 排除特定字符

使用**方括号**和 `^` 来排除特定字符。

**练习：排除字符**

```
Task	Text	 
Match	hog	
Match	dog	
Skip	bog

[^b]og
```

## Lesson 5: 字符范围

在使用方括号表示法时，可以通过使用破折号表示字符范围来简化顺序字符列表中的字符匹配。

例如，模式 `[0-6]` 将只匹配0到6的任何数字单位，不匹配其他任何字符。同样的，`[^n-p]` 将只会匹配除字母n到p之外的任何单个字符。

同一组括号里可以使用多个字符范围以及单个字符。这方面的一个例子是代表字符或数字的元字符 `\w` ,它等价于 `[A-Za-z0-9]`

**练习：字符范围匹配**

```
Task	Text	 
Match	Ana	
Match	Bob	
Match	Cpc	
Skip	aax	
Skip	bby	
Skip	ccz

[A-C][n-p][a-c]
```

## Lesson 6: 匹配重复字符

当需要匹配重复字符，一种方便的方法是使用花括号表示法指定每个字符的重复次数。例如，`{3}` 将精确匹配字符三次。某些正则表达式引擎甚至允许您为这种重复指定一个范围，以便 `{1,3}` 匹配一个字符不超过3次，但不少于一次。

这种表示法可以和任何字符或特殊字符使用，例如 `w{3}` `[wxy]{3}` `.{2,6}`。

**练习：匹配重复的字符**

```
Task	Text	 
Match	wazzzzzup	
Match	wazzzup	
Skip	wazup

waz{3,5}up
```

## Lesson 7: 量词元字符

使用 `*` 元字符来匹配任意数量的字符。

使用 `+` 元字符来匹配一个及以上的字符。

如果要匹配*和+字符，请使用 `\*` `\+`

这些限定符可以用于任何字符或特殊的元字符，例如 `a+` `[abc]+`。

**练习：匹配重复的字符**

```
Task	Text	 
Match	aaaabcc	
Match	aabbbbc	
Match	aacc	
Skip	a

aa+b*c+
```

## Lesson 8: 可选字符

在匹配和提取文本时另一个常见的量词是 `?` 来代表可选性的元字符。

此字符允许您匹配前面的字符或字符组0个或1个。

例如，模式 `ab?c` 将匹配字符 "abc" 和 "ac"，b被认为是可选的。

**练习：匹配可选字符**

```
Task	Text	 
Match	1 file found?	
Match	2 files found?	
Match	24 files found?	
Skip	No files found.

\d+ files? found\?
```
