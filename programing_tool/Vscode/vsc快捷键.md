## **通用**

**Ctrl+Shift+P/F1: 显示命令面板**

**Ctrl+P: 快速打开**

Ctrl+Shift+N: 新窗口/实例

**Ctrl+W: 关闭窗口/实例**

Ctrl+,: 用户设置

Ctrl+K Ctrl+S: 按键绑定




## 基础编辑

**Ctrl+X: 剪切行（空选择）**

**Ctrl+C: 复制行（空选择）**

Alt+ ↓ / ↑: 移动当前行

Ctrl+Shift+K: 删除行

**Ctrl+Enter / Ctrl+Shift+Enter: 在下方/上方插入一行（并且光标移到那一行）**

Ctrl+Shift+\\: 跳转到匹配的括号

Ctrl+] / Ctrl+[: 减少/增加缩进

Home / End: 回到行的开始或最后

Ctrl+ Home / End: 回到文件的开始或最后

**Ctrl+ ↑ / ↓: 视图向上向下滚动一行**

Alt+ PgUp / PgDn: 视图向上向下滚动一页

Ctrl+Shift+ [ / ]: 折叠/展开区域

Ctrl+K Ctrl+ [ / ]: 折叠/展开所有子区域

Ctrl+K Ctrl+0 / Ctrl+K Ctrl+J: 折叠/展开所有区域

Ctrl+K Ctrl+C: 添加行注释

Ctrl+K Ctrl+U: 移除行注释

Ctrl+/: 切换行注释

Ctrl+Shift+A: 切换块注释

Alt+Z: 切换单词换行



## 富语言编辑

**Ctrl+Space: 触发建议**

**Ctrl+Shift+Space: 触发参数提示**

**Ctrl+Shift+I: 格式化文档**

Ctrl+K Ctrl+F: 格式化选择的部分

**F12: 转到定义**

**Ctrl+Shift+F10: 窥视定义**

Ctrl+K F12: 将定义从一边打开

**Ctrl+.: 快速修复**

**Shift+F12: 显示引用**

**F2: 变量重命名**

Ctrl+K Ctrl+X: 修剪尾随空格

Ctrl+K M: 更改语言模式



## 多光标和选择

**Alt+Click 插入光标**

Shift+Alt+ ↑ / ↓ 往上/下方插入光标 

Ctrl+U 撤销上次的光标操作

Shift+Alt+I 在行尾添加光标

**Ctrl+L 选中当前行**

**Ctrl+Shift+L 选择所有查找到的匹配项**

Ctrl+F2 选择所有查找到的匹配单词

**Shift+Alt + → 展开选择**

**Shift+Alt + ← 收缩选择**

**Shift+Alt + drag mouse 框选择**



## 显示

**F11 切换全屏**

**Shift+Alt+0 切换编辑器布局（水平/垂直，必须要两个以上的窗口才有效）**

**Ctrl+ = / - 放大/缩小**

**Ctrl+B 切换侧边栏可见性**

Ctrl+Shift+E 显示资源管理器

Ctrl+Shift+F 显示搜索

Ctrl+Shift+G 显示源代码管理

Ctrl+Shift+D 显示Debug

Ctrl+Shift+X 显示扩展

Ctrl+Shift+H 在文件执行替换

Ctrl+Shift+J 切换搜索细节

**Ctrl+Shift+C 打开新的终端**

Ctrl+K Ctrl+H 显示输出面板

Ctrl+Shift+V 打开Markdown预览

**Ctrl+K V 打开Markdown预览并置放到一边**

## 搜索和替换

**Ctrl+F 搜索**

**Ctrl+H 替换**

**F3 / Shift+F3 查找下一个/上一个**

**Alt+Enter 使用光标选择所有搜索匹配项**

Ctrl+D 将下一个匹配项添加到选择

Ctrl+K Ctrl+D 将上次选择移动到下一个选择匹配项



## 导航

**Ctrl+W 关闭编辑窗口**

Ctrl+K Ctrl + ← 聚焦于上一个编辑器组
