# 过滤数据

## DISTINCT

在表的一列或多列中选择不同的值。

## WHERE

根据一个或多个条件过滤查询输出中的行。

## AND

组合两个布尔表达式，如果所有表达式都为真，则返回真。

## OR

组合两个布尔表达式，如果其中一个条件为真，则返回真。

## IN

检查值是否与列表或子查询中的任何值匹配。

## BETWEEN

测试一个值是否在一个值范围之间。

## LIKE

检查字符串是否与指定的模式匹配。

## 列的别名

向您展示如何使用列别名更改查询输出的标题和表别名以提高查询的可读性。
