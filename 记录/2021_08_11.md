## python类型转换

可以手动的执行类型转换。

| 类型转换函数    | 转换路径         |
| --------------- | ---------------- |
| `float(string)` | 字符串 -> 浮点值 |
| `int(string)`   | 字符串 -> 整数值 |
| `str(integer)`  | 整数值 -> 字符串 |
| `str(float)`    | 浮点值 -> 字符串 |



## python真值检测

检测真值的优雅方式是这样的：

```python
if x:
    pass
```

不要像下面这样做：

```python
if x == True:
    pass
```
