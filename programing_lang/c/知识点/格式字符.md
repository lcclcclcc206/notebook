## `printf` 和 `scanf` 中的格式字符

![5-2.5.3-1](../../../../../桌面/NoteBook/记录/.2021_07_31/document-uid18510labid311timestamp1527235892321.png)

> 小知识点：
>
> 1. 在输入函数时，用 %c 格式声明输入字符时，空格字符和转义字符都是作为有效字符输入，例如：`scanf("%c%c%c",&c1,&c2,&c3);` 在执行这个程序时，需要我们连续输入 3 个字符，中间不要有空格，如下： `abc` 下面插入空格的形式是错误的： `a b c`
>
> - 这种形式第一个字符 'a' 送给 c1，第二个字符是空格字符' '送给了 c2，第三个字符 'b' 送给了 c3。
>
> 1. 在使用 %d 输出时，我们可以指定输出的宽度。具体用法:
>
> - `%d`：按照整型数据的实际长度输出。
> - `%md`：以 m 为指定的最小字段宽度输出，右对齐。
> - `%ld`：输出长整型数据。
> - `%mld`：输出指定宽度的长整型数据。

```c
#include <stdio.h>

int main()
{
    int a = 12, b = -3456;
    long int c = 123456;
    printf("%5d\n", a);
    printf("%d\n", b);
    printf("%ld\n", c);
    printf("%6ld\n", c);

    return 0;
}
```
